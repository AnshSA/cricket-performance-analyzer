# 🏏 Cricket Performance Analyzer Architecture

## Project Overview

The Cricket Performance Analyzer is a full-stack web application that helps cricket players and coaches analyze performance using statistics, reports, and visual dashboards.

---

## System Architecture

```
Frontend
    │
    ▼
HTML • CSS • JavaScript
    │
    ▼
FastAPI Backend
    │
    ▼
Database
```

---
